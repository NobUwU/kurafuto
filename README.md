  <h1 align="center">Kurafuto</h1>
  <p>Kurafuto is a simple project itilizing Node.js and Vue to create a web interface for Element Zero's BDS console</p>
</p>

### How to Use

- Clone or download this repository
- Run `npm install`
- `npm run dev` for development enviorment with split api and frontend, `npm start` to build files for production and start server
